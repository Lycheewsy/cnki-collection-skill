# CNKI CSSCI 全文采集 Skill 使用说明

## 一、项目简介

本项目是一个可复用的 CNKI CSSCI 全文采集 Skill，用于根据知网专业检索式自动检索 CSSCI 期刊文献，并尝试提取可访问的 HTML 全文内容。成功提取的文献将保存为 TXT 文件，跳过或失败的文献将记录在 CSV 文件中，便于后续人工检查和语料库整理。

本项目适用于中文学术话语研究、术语翻译研究、文献综述、语料库构建等研究场景。

## 二、主要功能

本 Skill 目前支持以下功能：

1. 根据知网专业检索式进行文献检索；
2. 限制来源为 CSSCI 期刊；
3. 支持手动登录知网后继续自动采集；
4. 自动逐页读取检索结果；
5. 尝试打开文献详情页和 HTML 阅读页；
6. 提取可访问的 HTML 全文；
7. 将成功提取的全文保存为 TXT 文件；
8. 将无 HTML 全文、提取失败或跳过的文献记录到 CSV 文件中；
9. 通过 `run_skill.py` 作为统一入口运行。

## 四、运行环境

使用本项目需要：

1. Python 3；
2. Google Chrome 浏览器；
3. 合法可用的 CNKI 访问权限；
4. 终端或 VS Code 运行环境。

## 五、安装方法

进入项目目录后，建议先创建虚拟环境：

```bash
python3 -m venv .venv
source .venv/bin/activate
```

安装依赖：

```bash
python3 -m pip install -r requirements.txt
```

## 六、运行方法

推荐使用以下命令进行小批量测试：

```bash
python3 run_skill.py --max-pages 1 --delay-min 6 --delay-max 10
```

运行后，浏览器会自动打开知网页面。用户需要先手动完成知网登录，然后回到终端按回车，程序会继续执行后续采集流程。

## 七、输出结果

成功提取的 TXT 文件和失败记录会输出到：

```text
outputs/cnki_fulltext_output/
```

其中：

* `.txt` 文件：保存成功提取的文献全文；
* `failed_and_skipped_articles.csv`：记录无 HTML 全文、提取失败或被跳过的文献。

## 八、注意事项

1. 本项目不用于绕过知网权限、付费墙或访问限制；
2. 用户必须使用合法授权的 CNKI 访问方式；
3. 采集结果仅建议用于合规的个人学术研究和语料整理；
4. 连续采集大量文献时，Chrome 可能出现卡顿，建议小批量运行；
5. 若知网页面结构变化，部分 CSS 选择器或 XPath 可能需要更新。
